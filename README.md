# MercadoJusto
Mercado Justo - Checkpoint 2 Projeto desenvolvido em Java para monitoramento e análise de preços em diferentes estabelecimentos, com o objetivo de identificar disparidades de valores em produtos essenciais.

Nome dos Contribuintes - 

Funcionalidades
Cadastro de Produtos: Registro de itens em uma rede de 6 mercados (3 Populares e 3 Centrais).
Comparação de Preços: Localiza o menor preço para um produto específico entre todos os mercados.
Média por Produto: Calcula o valor médio de um item na região para identificar preços abusivos.

Poo
O projeto segue os princípios da Programação Orientada a Objetos:
Encapsulamento: Atributos privados com Getters e Setters.
Pacotes: Organizado em model, service e view.
Coleções: Uso de ArrayList para armazenamento dinâmico.

Pastas
src/model: Classes Produto e Estabelecimento.
src/service: Classe EstabelecimentoService (Lógica de comparação e média).
src/view: Classe Menu (Interface do sistema).
